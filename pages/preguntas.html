Fundamentos de Desarrollo Web

¿Qué es HTML y cuál es su función principal?

HTML (HyperText Markup Language) es el lenguaje utilizado para estructurar el contenido de una página web. Su función principal es organizar elementos como encabezados, párrafos, imágenes y enlaces.

¿Qué son las etiquetas HTML y cómo se utilizan?

Las etiquetas HTML estructuran el contenido de una página. Se escriben con una apertura <etiqueta> y un cierre </etiqueta>.

Ejemplos de etiquetas HTML comunes

  • <p> - Párrafo
  • <h1> a <h6> - Encabezados
  • <a href="url"> - Enlace
  • <img src="imagen.jpg"> - Imagen

¿Qué es CSS y cómo se utiliza?

CSS (Cascading Style Sheets) es un lenguaje que permite personalizar el diseño de una página web, como los colores, las fuentes y las posiciones.

Ejemplo de CSS:

body { background-color: #f4f4f4; color: #333; }

¿Qué es el modelo de caja (box model)?

El modelo de caja define cómo los elementos se estructuran en la web. Sus partes son:

  • Contenido: Área donde se muestra el texto o imágenes.
  • Padding: Espacio interno alrededor del contenido.
  • Border: Borde del elemento.
  • Margin: Espacio externo alrededor del borde.

¿Qué es el diseño responsivo?

El diseño responsivo adapta la página a diferentes pantallas utilizando @media queries y unidades relativas.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web.

Ejemplo de interacción con JavaScript:

document.getElementById("demo").innerHTML = "Texto cambiado";

¿Qué es Git y por qué es importante?

Git es un sistema de control de versiones que permite rastrear cambios y colaborar en proyectos de desarrollo web.

Comandos básicos de Git:

  • git init - Inicializa un repositorio.
  • git add . - Agrega cambios.
  • git commit -m "Mensaje" - Guarda cambios.
  • git push origin main - Sube cambios al repositorio remoto.

¿Qué es GitHub?

GitHub es una plataforma que permite almacenar repositorios Git en la nube y facilita la colaboración en proyectos web.
